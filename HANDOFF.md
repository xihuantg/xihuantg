# 豫志通项目交接文档

## 1. 项目概况

项目名：豫志通

定位：面向河南新高考 3+1+2 考生和家长的志愿填报辅助工具。当前是静态前端展示版，不接真实后端、不接真实支付、不接真实 AI API。

目标用户：河南优先，后续扩展山东、河北、山西等山河四省家庭。

核心卖点：

- 根据河南 3+1+2 选科、分数、位次、批次、城市和专业偏好生成冲稳保推荐。
- 不只给学校名单，还解释选科、位次、专业组风险、就业、考研、学习难度、转专业路径。
- 低价线上报告型产品，不做线下一对一专家服务。
- 重点解决普通家庭怕浪费分、怕滑档、怕专业坑、家里讨论不清楚的问题。

## 2. 当前功能

### 桌面端

- 首页诊断表单：河南、首选科目、再选科目、分数、位次、批次、专业偏好、城市偏好。
- 智能推荐：按冲、稳、保、不可报展示院校专业组。
- 志愿表：可加入、排序、删除，免费版限制数量。
- 风险诊断：检查选科不匹配、保底不足、冲刺过多、热门专业集中等问题。
- 院校专业组库：显示选科要求、位次、就业/难度摘要、可报状态。
- 专业目录：93 个专业类，支持搜索和门类筛选。
- 学校详情页：官网、校园照片占位、录取趋势、选科要求、就业升学、适合画像、学习难度、转专业与校内路径。
- 会员套餐：基础版 29 元 / 本届高考季，进阶版 59 元 / 本届高考季。
- AI 性格与职业倾向测评：10 道选择题，生成画像、适合方向、专业和院校专业组推荐。

### 手机端

- App 风格首页。
- 固定底部导航：首页、推荐、院校、诊断、我的。
- 首页工具入口：智能推荐、性格测评、志愿诊断、院校库、专业目录、会员权益。
- 推荐页、院校页、诊断页、我的页分区显示，避免长页面堆叠。
- 性格测评归入诊断页。
- 推荐学校可进入详情页。

## 3. 付费设计

### 免费版

- 部分推荐结果。
- 基础考生档案。
- 少量院校专业组信息。
- 基础性格画像和少量匹配专业组。

### 基础版：29 元 / 本届高考季

- 完整冲稳保名单。
- 选科可报检查。
- 基础风险等级。
- 就业方向和学习难度摘要。
- 收藏并加入志愿表。
- 院校专业组基础详情。

### 进阶版：59 元 / 本届高考季

- 基础版全部权益。
- 推荐理由和排序说明。
- 替代院校建议。
- 专业组深度风险。
- 考研升学、适合人群和大学四年学习建议。
- 转专业可行性与校内提升路径。
- 家长版解释报告。
- 导出/打印与多次模拟。

注意：当前只是模拟会员解锁，不接真实支付。

## 6. 重要注意事项

### 编码问题

之前出现过中文变成 `????` 的问题。

原因：在 Windows PowerShell 里通过管道或脚本直接写中文，编码可能被转换，导致写入文件前中文已经变成问号。

以后改中文时：

- 优先用 `apply_patch`。
- 或使用 HTML 数字实体。
- 不要用 PowerShell 管道直接写大段中文。
- 修改后必须扫描 `??`：

```powershell
Select-String -Path index.html,data.js,app.js,styles.css -Pattern '\?\?+'
```

### 数据真实性

当前所有院校、分数、位次、就业、环境评价、转专业说明都是展示版样板数据。

正式收费前必须替换为真实、可核验、可合法使用的数据，并补充来源：

- 河南省教育考试院。
- 高校招生章程。
- 高校本科生手册 / 教务处转专业政策。
- 高校官网。
- 合法可引用的公开评价摘要。

### 不做的内容

当前阶段不做：

- 真实支付。
- 真实登录。
- 后端数据库。
- 真实 AI API。
- 线下一对一专家服务。
- 保录取、内部数据、百分百准确等承诺。
